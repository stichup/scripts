# Warning

# Usage
xxhdpi-to-dpis.sh creates resampled copies of an **xxhdpi_file** image to xhdpi, hdpi, mdpi, and ldpi images. The command should be composed to be like this:

	./xxhdpi-to-dpis.sh <xxhdpi_file>
